Hi!

I'm a game programmer who is passionate about learning everything I can about game programming. I like to understand what goes on under the hood, even when using ready made engines like Unity or Unreal. I believe there are still many things to achieve in game development beyond what ready made engines are cabable of currently. Projects

Here is a list of some projects I've been a part of:

The Land of the Scurvy Dog

A 3D platformer demo made with Unreal Engine 4. You play as a pirate corgi. The game focuses on smooth controls and challenging platforming. A full version of the game is currently in development.
What I did in this project:
  • Gameplay code (Controls, camera stuff, ...)
  • Game flow and systems (checkpoints, score counting, lose states, ...)
  • Polish stuff (effects, feedback, ...)
  • All sorts of other coding and tech stuff

Gravity Creatures

A gravity based mobile puzzle game with 40 levels divided into 4 biomes, each biome introducing a new gimmick. The game was made with Unity for Android during the summer of 2016.
What I did in this project:
  • Most of the UI code
  • Content loading and management
  • Gameplay code and polish stuff

Insatiable

A fast-paced top down shooter with one playable demo level and two enemy types.
What I did in this project:
  • Gameplay code mostly

Sonar Bat (GGJ17)

Sonar Bat was our Global Game Jam 2017 entry. It's a flappy bird like game with a little twist. You can only see when you tap the screen. Bats are blind after all. It is available for download on Google Play.

Insult to Injury

A game about coming up with the best insults. The goal of the game is to insult your opponent in the worst ways. The game features online and local multiplayer. You can also take a shot at beating the AI opponent. The game was made with Unity for Android during the spring of 2016.
What I did in this project:
  • Implementing online features (match-making, cloud save, ...)
  • Content loading and management (word packs, hats)
  • Gameplay implementation
  • Bunch of little polish
